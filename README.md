# Mini Editeur TP2-OMD

## V1

### Fonctionnalités

- Ouverture de fichier existant
- Création de fichier existant
- Déplacement du curseur horizontalement et verticalement
- Insertion de caractères
- Insertion de nouvelle ligne
- Suppression de caractères arrière (backspace) et avant (del)
- Suppression de ligne 
- Copie presse papier 
- Insertion de copie 
- Sélection de texte horizontalement et verticalement
- Sauvegarde de fichier
- Sélection de tout le fichier

### Commandes utilisateur

#### Sélection du mode

- Mode Normal : Esc
- Mode Sélection : CTRL+S

#### Édition
- Déplacement du curseur horizontalement et verticalement : KEY UP/DOWN/LEFT/RIGHT
- Insertion de caractère: Mode Normal + caractère
- Insertion de nouvelle ligne : ENTER
- Copie de sélection : Mode Sélection + CTRL+C
- Insertion de sélection : Mode Sélection + CTRL+V
- Suppression caractère précédent : BACKSPACE
- Suppression caractère suivant : DEL
- Suppression de ligne : CTRL+K
- Sélection de tout le contenu : CTRL+A
- Sauvegarder : CTRL+W
- Quitter : CTRL+Q
